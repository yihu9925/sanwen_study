# 路由与导航设计 — @kit.ArkUI router

> ⚠ API 20：导入使用 `@kit.ArkUI`，支持命名路由传参

## 导入方式

```typescript
import { router } from '@kit.ArkUI';

// 带参数跳转
router.pushUrl({
  url: 'pages/CourseDetailPage',
  params: { courseId: '...' }
});

// 返回
router.back();
```

## 页面层级结构

```
Index.ets (应用主入口 - 底部3Tab容器)
+-- HomePage (Tab1: 首页)
|   +-- CourseDetailPage (详情页, @Entry独立路由)
|       +-- [Tabs内] Q1_KnowledgeGraphPage
|       +-- [Tabs内] Q2_ControversyPage
|       +-- [Tabs内] Q3_AssessmentPage
|       |   +-- AssessmentDetailPage (答题详情页 - @Entry独立路由)
|       +-- [Tabs内] 资料库文档列表
|           +-- DocSheet (CustomDialog弹窗)
+-- KnowledgeHub (Tab2: 知识库总览, 经由KnowledgeHubHost装载)
|   +-- DocSheet (CustomDialog弹窗显示文档正文)
+-- PersonalCenter (Tab3: 个人中心, 经由PersonalCenterHost装载)
    +-- ArchiveManagementPage (归档管理 - @Entry独立路由)
    +-- SettingsPage (设置 - @Entry独立路由)
        +-- AppearancePage (外观 - @Entry独立路由)
```

## 应用主入口: Index.ets (替代原MainLayout)

```typescript
// Index.ets — 应用主入口，3个底部Tab
import { PersonalCenterHost } from './PersonalCenterHost';
import { KnowledgeHubHost } from './KnowledgeHubHost';
import { HomePage } from './HomePage';

@Entry
@Component
struct Index {
  @StorageLink('activeTabIndex') activeTabIndex: number = 0;
  @StorageLink('themeMode') themeMode: string = 'auto';
  @StorageLink('uiStyle') uiStyle: string = 'default';
  @State homePageKey: number = 0;
  @State personalCenterKey: number = 0;
  @State knowledgeHubKey: number = 0;

  aboutToAppear(): void {
    AppStorage.setOrCreate('uiStyle', 'default');
  }

  onPageShow(): void {
    // 从推送页面返回时，强制所有 tab 重建以刷新数据
    CourseService.refreshStatsInStorage('default_user').then(() => {
      this.homePageKey++;
      this.personalCenterKey++;
      this.knowledgeHubKey++;
    });
  }

  build() {
    Tabs({ barPosition: BarPosition.End, index: this.activeTabIndex }) {
      TabContent() {
        HomePage().key('hp_' + this.homePageKey);
      }
      .tabBar('\u9996\u9875');  // 首页

      TabContent() {
        KnowledgeHubHost().key('kh_' + this.knowledgeHubKey);
      }
      .tabBar('\u77E5\u8BC6\u5E93');  // 知识库

      TabContent() {
        PersonalCenterHost().key('pc_' + this.personalCenterKey);
      }
      .tabBar('\u4e2a\u4eba');  // 个人
    }
    .onChange((index: number) => {
      this.activeTabIndex = index;
      // 切换Tab时强制刷新对应内容
    });
  }
}
```

### Tab切换刷新机制

Tab切换时通过递增`.key()`值强制重建组件，避免ArkUI的@State缓存问题：

- **首页Tab** (index=0): 切换时递增 homePageKey
- **知识库Tab** (index=1): 切换时递增 knowledgeHubKey
- **个人Tab** (index=2): 先刷新统计数据再递增 personalCenterKey

```typescript
.onChange((index: number) => {
  this.activeTabIndex = index;
  if (index === 0) { this.homePageKey++; }
  if (index === 1) { this.knowledgeHubKey++; }
  if (index === 2) {
    CourseService.refreshStatsInStorage('default_user').then(() => {
      this.personalCenterKey++;
    });
  }
});
```

## 路由表

```typescript
// AppRouter.ts — 完整路由配置
import { router } from '@kit.ArkUI';

interface RouteConfig {
  HOME: string;
  COURSE_DETAIL: string;
  ASSESSMENT_DETAIL: string;
  ARCHIVE: string;
  SETTINGS: string;
  Q1_KNOWLEDGE_GRAPH: string;
  Q2_CONTROVERSY: string;
  Q3_ASSESSMENT: string;
  APPEARANCE: string;
}

export const ROUTES: RouteConfig = {
  HOME: 'pages/HomePage',
  COURSE_DETAIL: 'pages/CourseDetailPage',
  ASSESSMENT_DETAIL: 'pages/AssessmentDetailPage',
  ARCHIVE: 'pages/ArchiveManagementPage',
  SETTINGS: 'pages/SettingsPage',
  Q1_KNOWLEDGE_GRAPH: 'pages/Q1_KnowledgeGraphPage',
  Q2_CONTROVERSY: 'pages/Q2_ControversyPage',
  Q3_ASSESSMENT: 'pages/Q3_AssessmentPage',
  APPEARANCE: 'pages/AppearancePage',
};

// 导航工具类
export class AppRouter {
  static toCourseDetail(courseId: string): void { ... }
  static toAssessmentDetail(courseId: string): void { ... }
  static toQ1KnowledgeGraph(courseId: string): void { ... }
  static toQ2Controversy(courseId: string): void { ... }
  static toQ3Assessment(courseId: string): void { ... }
  static toArchive(): void { ... }
  static toSettings(): void { ... }
  static toAppearance(): void { ... }
  static back(): void { ... }
}
```

## 页面参数传递

| 源页面 | 目标页面 | 参数 | 说明 |
|-------|---------|------|------|
| HomePage | CourseDetailPage | `{ courseId: string }` | 点击课程卡片 |
| CourseDetailPage | Q1_KnowledgeGraphPage | `{ courseId: string }` | 查看知识图谱详情 |
| CourseDetailPage | Q2_ControversyPage | `{ courseId: string }` | 查看争议点详情 |
| CourseDetailPage | Q3_AssessmentPage | `{ courseId: string }` | 查看测评题列表 |
| Q3_AssessmentPage | AssessmentDetailPage | `{ courseId: string }` | 开始自由答题 |
| PersonalCenter | ArchiveManagementPage | — | 查看归档 |
| PersonalCenter | SettingsPage | — | 设置 |
| SettingsPage | AppearancePage | — | 外观设置 |
| KnowledgeHub | CourseDetailPage | `{ courseId: string }` | 从文档弹窗跳转课程 |

## 获取页面参数

```typescript
import { router } from '@kit.ArkUI';

@Entry
@Component
struct CourseDetailPage {
  @State courseId: string = '';

  aboutToAppear(): void {
    const params = router.getParams() as Record<string, string>;
    this.courseId = params?.['courseId'] || '';
    if (this.courseId) {
      this.loadCourseData();
    }
  }
}
```
