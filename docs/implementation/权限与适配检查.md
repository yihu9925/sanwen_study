## 检查清单

### 1. 版本兼容性
- [ ] compileSdk = 20
- [ ] compatibleSdk = 20
- [ ] 使用 @kit.* 命名空间
- [ ] 所有 import 已检查是否支持 API 20

### 2. 页面适配
- [ ] 手机 360vp-480vp 宽度测试
- [ ] 平板 600vp-1600vp 宽度测试
- [ ] ResponsiveContainer 使用 `breakpoints` 断点
- [ ] 横竖屏切换正常
- [ ] 键盘弹出时页面不遮挡输入框

### 3. 深色模式
- [ ] 使用 `sys.color.*` 系统色值
- [ ] 自定义颜色使用 `@Provider`/`@Consume` 主题传递

### 5. 错误处理
- [ ] 所有 async 函数有 try-catch
- [ ] 网络请求超时处理（timeout: 10000）
- [ ] 数据库操作有 fallback
- [ ] UI 层显示错误提示

### 6. 安全
- [ ] API Key 存储在 Preferences（加密）
